My name is Luke Venkataramanan, and I graduated from the University of Texas at Austin in May 2026, where I studied Computational Engineering. Starting in June 2026, I will be a software engineer at SpaceX.

## Highlighted Projects
### Premier League Predictor
[Premier League Predictor](https://github.com/luke-venk/premier-league-predictor) is a project that combines a custom machine learning model I made with partners in a class project with a full-stack web application I worked on independently. The app lets users run simulations that, depending on what games have been played in the 25-26 Premier League Season, predicts the probability of each possible outcome (home team wins, draw, away team wins), as well as the standings resulting from each of these match predictions.

### Mjölnir
[Mjölnir](https://github.com/luke-venk/mjolnir) was a project for senior design in which the team aimed to determine with high precision where a shot put thrown by an athlete landed. My work for this project was primarily architecting a computer vision pipeline which could run stages (e.g. lens undistortion, Mog2) in parallel, in order to make our system faster. I also spent many hours developing tools to gather footage from the cameras and implementing the Bazel build system for our project.  

[![My Skills](https://skillicons.dev/icons?i=rust,ts)](https://skillicons.dev)

### TREL
For several years, I was part of the Texas Rocket Engineering Lab at UT Austin, where I developed both flight software and ground software for a large liquid-bipropellant rocket. Due to International Traffic in Arms Regulation (ITAR), I cannot display the software. However, it's what I spent the vast majority of my extracurricular time on in college, and I have held several leadership positions with the lab.
