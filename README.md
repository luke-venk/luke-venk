## Highlighted Projects
### Premier League Predictor
[Premier League Predictor]([url](https://github.com/luke-venk/premier-league-predictor)) is a project that combines a custom machine learning model I made with partners in a class project with a full-stack web application I worked on independently. The app lets users run simulations that, depending on what games have been played in the 25-26 Premier League Season, predicts the probability of each possible outcome (home team wins, draw, away team wins), as well as the standings resulting from each of these match predictions.

### Mjölnir
[Mjölnir]([url](https://github.com/luke-venk/mjolnir)) was a project for senior design in which the team aimed to determine with high precision where a shot put thrown by an athlete landed. This required a performant backend (Rust), a well-developed frontend (TypeScript), and a lot of work with GigE Vision cameras. To summarize the system, we had 2 very high quality cameras connected over Ethernet to a switch, which also connected to our laptop and an Arduino system (to detect whether or not an athlete stepped out of bounds). My work for this project was primarily architecting a computer vision pipeline which could run stages (e.g. lens undistortion, Mog2) in parallel, in order to make our system faster. I also spent many hours developing tools to gather footage from the cameras and implementing the Bazel build system for our project.
[![My Skills](https://skillicons.dev/icons?i=rust,ts)](https://skillicons.dev)
